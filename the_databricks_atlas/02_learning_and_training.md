# Learning & Training: Level Up Your Databricks Skills

This page provides a curated list of the best resources for learning Databricks, from guided tutorials to official certifications.

## Official Databricks Training

- **[Databricks Academy](https://www.databricks.com/learn/academy)**: The official source for Databricks training and certification. Offers a wide range of courses, from beginner to advanced.
- **[Databricks Certifications](https://www.databricks.com/learn/certification)**: Validate your skills with official Databricks certifications, including Data Engineer, Machine Learning Engineer, and more.
- **[Databricks Demos](https://www.databricks.com/resources/demos)**: A collection of live and on-demand demos showcasing various Databricks features and use cases.

## Free Learning Resources

- **[Databricks Community Edition](https://community.cloud.databricks.com/)**: A free, limited-use version of the Databricks platform, perfect for learning and personal projects.
- **[Databricks Tutorials](https://docs.databricks.com/en/getting-started/tutorials/index.html)**: A comprehensive set of tutorials from the official documentation.
- **[DataCamp: Learn Databricks](https://www.datacamp.com/courses/introduction-to-databricks)**: An interactive course from DataCamp that covers the fundamentals of Databricks.

## Recommended Courses

- **[Udemy: Azure Databricks & Spark for Data Engineers](https://www.udemy.com/course/azure-databricks-spark-for-data-engineers/)**: A highly-rated course that covers the essentials of Azure Databricks and Spark.
- **[Coursera: Databricks for Data Engineering](https://www.coursera.org/professional-certificates/databricks-for-data-engineering)**: A professional certificate program from Coursera that provides a deep dive into data engineering with Databricks.
