# Solution Accelerators: Production-Ready Blueprints

Databricks Solution Accelerators are fully functional notebooks that tackle the most common and high-impact use cases. They are vetted and built by industry experts at Databricks and serve as a great starting point for new projects.

## Key Solution Accelerator Repositories

- **[Databricks Industry Solutions](https://github.com/databricks-industry-solutions)**: The main hub for industry-specific solution accelerators, organized by vertical (Financial Services, Healthcare, Retail, etc.).
- **[Databricks Solution Accelerators Website](https://www.databricks.com/solutions/accelerators)**: The official website with a searchable catalog of all solution accelerators.

## Popular Solution Accelerators

- **[Security Analysis Tool](https://github.com/databricks-industry-solutions/security-analysis-tool)**: Analyzes your Databricks account and workspace security configurations and provides recommendations.
- **[DIY LLM QA Bot](https://github.com/databricks-industry-solutions/diy-llm-qa-bot)**: A guide to building a question-answering bot using Large Language Models.
- **[Many Model Forecasting](https://github.com/databricks-industry-solutions/many-model-forecasting)**: A project for bootstrapping large-scale forecasting solutions.
- **[ESG Scoring](https://github.com/databricks-industry-solutions/esg-scoring)**: An approach to sustainable finance using NLP and news analytics.
- **[Smolder (HL7 Parser)](https://github.com/databricks-industry-solutions/smolder)**: An Apache Spark data source for parsing HL7 v2 messages.
