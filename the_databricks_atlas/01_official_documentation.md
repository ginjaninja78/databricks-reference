# Official Documentation: The Source of Truth

This page contains direct links to the official documentation for the core components of the Databricks ecosystem. When in doubt, these are the ultimate sources of truth.

## Databricks Documentation

- **[Databricks on AWS](https://docs.databricks.com/aws/index.html)**: The main documentation hub for Databricks on AWS.
- **[Databricks on Azure](https://docs.microsoft.com/en-us/azure/databricks/)**: The main documentation hub for Azure Databricks.
- **[Databricks on GCP](https://docs.gcp.databricks.com/)**: The main documentation hub for Databricks on Google Cloud.
- **[Databricks Best Practices](https://docs.databricks.com/en/getting-started/best-practices.html)**: A curated collection of best practice guides from Databricks.
- **[Databricks Release Notes](https://docs.databricks.com/en/release-notes/index.html)**: Stay up-to-date with the latest features and changes.

## Delta Lake Documentation

- **[Delta Lake Official Site](https://delta.io/)**: The home of the Delta Lake project.
- **[Delta Lake Documentation](https://docs.delta.io/latest/index.html)**: The official documentation for Delta Lake, covering everything from basic concepts to advanced features.

## Apache Spark Documentation

- **[Apache Spark Official Site](https://spark.apache.org/)**: The home of the Apache Spark project.
- **[Spark Documentation](https://spark.apache.org/docs/latest/)**: The official documentation for Apache Spark, including programming guides, API references, and more.

## MLflow Documentation

- **[MLflow Official Site](https://mlflow.org/)**: The home of the MLflow project.
- **[MLflow Documentation](https://mlflow.org/docs/latest/index.html)**: The official documentation for MLflow, covering all components: tracking, models, projects, and the model registry.
